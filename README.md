#
## Install packages
Open your terminal on the root of folder.

```bash
npm i
# or
yarn install
```


## Start API
First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:8123](http://localhost:8123) with your browser to see the result.
